Política de Privacidade

Este aplicativo foi desenvolvido por Emiliano Callegari. Ele tem o propósito de ajudar na visualização dos dividendos que você poderá receber de fundos imobiliários, centralizando essas informações de forma prática.

Dados do Usuário

O aplicativo não coleta, armazena ou compartilha nenhum dado pessoal do usuário. Todas as informações apresentadas são baseadas nas entradas fornecidas diretamente pelo usuário no próprio dispositivo.

Uso de Dados

Nenhum dado sensível ou pessoal é processado pelo aplicativo. As informações exibidas sobre fundos imobiliários são de uso exclusivo para consulta e organização do usuário.

Isenção de Responsabilidade

Os fundos imobiliários listados no aplicativo não constituem qualquer tipo de recomendação de investimento. O objetivo é apenas informar e organizar os dividendos visualizados.
